隐私政策 (Privacy Policy)

本程序(zSMTH-Android)尊重并保护所有使用服务用户的个人隐私权。

1. 第三方SDK使用说明 (Third-party SDKs)

  • ShareSDK: 用以分享内容到微信、微博等第三方平台。
  • UMeng SDK: 用以进行应用版本检测和基本使用情况分析（用于改善应用体验）。

2. 权限使用说明 (Permissions)

本程序需要申请以下权限以正常运行：

  • android.permission.INTERNET: 访问网络，获取论坛数据。
  • android.permission.ACCESS_NETWORK_STATE: 检测网络状态（WiFi/移动数据）。
  • android.permission.ACCESS_WIFI_STATE: 获取WiFi状态。
  • android.permission.READ_EXTERNAL_STORAGE: 读取存储卡（用于上传附件/图片）。
  • android.permission.WRITE_EXTERNAL_STORAGE: 写入存储卡（用于保存图片/附件、缓存数据）。
  • android.permission.VIBRATE: 震动反馈（辅助提醒功能）。

3. 隐私声明 (Privacy Statement)

本程序不会收集任何用户的个人敏感隐私信息。应用的登录、发帖等操作均为用户与水木社区服务器的直接交互，本程序仅作为客户端工具。

4. 联系方式 (Contact)

如有任何疑问，请通过 GitHub Issues 联系我们。
